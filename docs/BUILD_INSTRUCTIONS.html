Quarto Build Instructions for Rishav’s Website

Installation

Quarto installation can be complex. Here are the recommended approaches:

Option 1: Download from Official Website (Recommended)

  1. Visit: https://quarto.org/docs/get-started/
  2. Download the macOS (arm64) installer for your system
  3. Follow the installation wizard

Option 2: Using Homebrew (Easiest)

brew install quarto

Option 3: Download Latest Release Directly

# For macOS ARM64 (Apple Silicon)
cd /tmp
curl -L -o quarto.tar.gz https://github.com/quarto-dev/quarto-cli/releases/download/v1.8.26/quarto-1.8.26-macos-aarch64.tar.gz
tar -xzf quarto.tar.gz
sudo mv quarto-*/bin/quarto /usr/local/bin/

Building the Site

Once Quarto is installed:

cd /Users/rishav/website/drganghe.github.io

# Render the entire site
quarto render

# Or for preview mode with live reload
quarto preview

The built site will be generated in the docs/ directory.

Troubleshooting

If you encounter issues:

  1. Quarto not found: Ensure installation is complete: quarto --version
  2. Image paths broken: Check that image URLs are using /files/images/ format
  3. Posts not showing: Ensure posts.qmd exists and lists the posts directory
  4. Build errors: Check console output for specific file/path issues
