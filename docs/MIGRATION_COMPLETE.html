File Conversion

Jekyll → Quarto Format Changes

Posts: - Jekyll: _posts/YYYY-MM-DD-title.md - Quarto: posts/YYYY-MM-DD-title/index.qmd

Pages: - Jekyll: _pages/about.md (with Jekyll-specific YAML) - Quarto: about.qmd (with Quarto-specific YAML)

Images: - Jekyll: assets/img/ - Quarto: files/images/

Bibliography: - Jekyll: _bibliography/papers.bib - Quarto: references.bib

Building the Site

To build and deploy your site:

  1. Install Quarto (if not already done):

    brew install quarto
# OR download from https://quarto.org/docs/get-started/

  2. Build the site:

    cd /Users/rishav/website/drganghe.github.io
quarto render

  3. Preview locally:

    quarto preview
# Opens at http://localhost:3456

  4. Deploy to GitHub Pages:

    • Push the repository to GitHub
    • In GitHub Pages settings, set source to “docs” directory
    • Site will be published at https://rish-av.github.io
